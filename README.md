# PersonReconizition_VS2015_Qt
基于VS2015+Qt开发的行人跟踪监测平台
=================================
* 项目背景
这是本人本科毕业设计的一部分，这个平台是基于VS+Qt+opencv开发的用于行人跟踪监测的小平台。
这种方法准确率并不高，开发目的是为了对比突出DPM算法的高准确率。
* 功能简介
这个小平台没有多少亮点，本人只是为了温习一下Qt的一些知识以及opencv的一些操作。
1.多界面编程，使用SIGNAL新号释放机制
2.文件名跑马效果
